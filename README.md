# Blog CMS Platform

Content management system for blogging with markdown support, image uploads, categories, and SEO optimization.

## Features

- Create, edit, delete blog posts
- Markdown editor with live preview
- Image upload and management
- Category and tag system
- SEO optimization
- User authentication
- Comment system
- Search functionality
- Responsive design

## Tech Stack

- Node.js / Express
- MongoDB
- React.js
- Markdown parser
- Multer (file uploads)

## Installation

```bash
npm install
npm run dev
```
